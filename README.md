# Proyecto Base de Datos – Tienda Online

## Descripción
Este proyecto demuestra la implementación de un sistema de gestión de base de datos relacional para una tienda en línea. Incluye la creación de tablas, relaciones, consultas y manipulación de datos.

## Contenido del Repositorio
- **ERD/**: Diagrama entidad-relación del modelo de datos.
- **SQL/DDL**: Scripts para crear y modificar tablas.
- **SQL/DML**: Scripts para insertar, actualizar y eliminar datos.
- **SQL/Consultas**: Consultas básicas y complejas para obtener información.
- **README.md**: Este archivo con explicación del proyecto.

## Cómo ejecutar
1. Abrir MySQL Workbench.
2. Ejecutar `crear_tablas.sql` para crear la base de datos y tablas.
3. Ejecutar los scripts de DML para poblar las tablas.
4. Ejecutar las consultas en el archivo `consultas_basicas.sql` y `consultas_complejas.sql` según se necesite.
